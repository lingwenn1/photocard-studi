# Photocard Studio

Веб-приложение для создания фотокарточек: степлер-скруглитель углов,
картхолдеры с голографическими эффектами, стикеры, текст, фильтры и экспорт.

## Стек

- **Next.js 15** (App Router) + **React 18** + **TypeScript**
- **Tailwind CSS** — тема, glassmorphism, тёмный/светлый режим
- **Framer Motion** — анимации UI, степлер, картхолдеры
- **GSAP** — анимация "клюва" степлера (сжатие/срез)
- **Konva / react-konva** — редактируемый холст (перемещение, масштаб,
  поворот, срез углов через `clipFunc`, стикеры, текст)
- **Web Audio API** (`lib/audio-engine.ts`) — звук степлера синтезируется
  программно (осцилляторы + шумовой импульс), без внешних mp3-файлов
- **zustand** — состояние редактора + история undo/redo
- **next-themes** — переключение тёмной/светлой темы
- **jsPDF** — экспорт в PDF
- **heic2any** — конвертация HEIC → JPEG на клиенте перед загрузкой в редактор

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

Продакшн-сборка:

```bash
npm run build
npm run start
```

Сборка проверена локально (`npm run build` проходит без ошибок).
Для сборки нужен доступ к `fonts.googleapis.com` (next/font/google
скачивает шрифты во время build) — это стандартное поведение Next.js
и не требует действий с вашей стороны при обычном деплое (Vercel и
большинство CI имеют доступ к интернету).

## Что реализовано полностью

- Загрузка JPG / PNG / WEBP / HEIC (HEIC конвертируется в браузере)
- Перемещение / масштаб / поворот фото на холсте (drag + Konva Transformer),
  плюс точные слайдеры
- **Степлер**: клик по любому углу → корпус подъезжает (Framer Motion) →
  челюсть сжимается (GSAP) → звук (Web Audio) → угол физически "срезается"
  через Konva `clipFunc` с индивидуальным радиусом на каждый угол,
  регулируемым слайдером
- **15 картхолдеров** — прозрачный, глянцевый, матовый, с блёстками,
  переливающийся, радужный, голографический, сердечки, звёзды, k-pop,
  kawaii, чёрный/белый минимализм, цветной пластик, с рамкой. Анимация
  "заезда" фото + появление плёнки поверх
- **Настоящий iridescent-эффект** на Canvas 2D (`holographic-overlay.tsx`):
  градиент реально смещается за курсором мыши, плюс твинклящие частицы
  для варианта "с блёстками"
- **11 принтов плёнки** (сердечки, звёзды, бабочки, облака, цветочки,
  котики, аниме, glitter, holographic, rainbow, sparkles)
- **Стикеры** — 14 категорий, перемещение/поворот/масштаб через Konva,
  копирование и удаление
- **Текст** — 6 шрифтов, цвет, размер, обводка, тень
- **Фон холста** — белый/чёрный/2 градиента/клетка/сердечки/звёзды/облака
- **Фильтры** — Vintage/VHS/Film/Polaroid/Soft/Bright/Dreamy/Warm/Cold +
  ручные яркость/контраст/насыщенность/резкость
- **Undo/Redo** — полная история состояния редактора
- **Экспорт** — PNG/JPG/WEBP (через `stage.toDataURL`) и PDF (jsPDF),
  с выбором качества и множителя разрешения
- Тёмная/светлая тема, мобильная адаптация (нижний тулбар с горизонтальным
  скроллом, панели инструментов в столбец на узких экранах)

## Осознанные упрощения (важно для честности)

- **Текстуры картхолдеров и принтов** реализованы параметрически —
  CSS-градиенты, canvas-шейдеры и эмодзи-паттерны, а не уникальные
  растровые PNG-ассеты для каждого из 15+11 вариантов. Визуально это
  передаёт стиль каждого типа и полностью интерактивно, но не является
  фотореалистичной текстурой пластика/фольги. При необходимости легко
  заменить `swatch`/`baseTint` в `lib/cardholders.ts` на реальные
  изображения (`background-image: url(...)`).
- **Резкость** в фильтрах — приближённая (через контраст), настоящая
  unsharp-mask свёртка на Canvas не подключена (это отдельная и тяжёлая
  операция, которую логично делать через WebGL-шейдер при развитии проекта).
- Звук степлера — синтетический, не запись реального устройства.
